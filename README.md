## Project Setup

### `Create .env file in Root Folder`
 ```javascript

    REACT_APP_LOGIN_URI=''
    REACT_APP_REQUEST_URI=''

 ```

### `npm install`
For installing necessary packages

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
